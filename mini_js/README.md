MiniJS Runtime
A lightweight JavaScript Runtime built entirely in Python.

MiniJS reads JavaScript source code, tokenizes it, parses it into an Abstract Syntax Tree (AST), and executes it through a custom interpreter without relying on Node.js, V8, Deno, Bun, or any JavaScript engine.

Built for the Build Your Own JavaScript Hackathon.

Requirements

Installation
Clone the repository:

git clone https://github.com/YOUR_USERNAME/mini-js-runtime.git
Move into the project:

cd mini-js-runtime
Install dependencies:

pip install -r requirements.txt
Running JavaScript Files
Execute a JavaScript file
Example:

python -m mini_js.main mini_js/examples/tc1_odd_even.js
Expected Output:

7 is Odd
Execute another example
python -m mini_js.main mini_js/examples/tc4_array_reverse.js
Expected Output:

Original: 1, 2, 3, 4, 5
Reversed: 5, 4, 3, 2, 1
Execute JavaScript directly from command line
python -m mini_js.main --code "let a = 5; let b = 10; console.log(a + b);"
Expected Output:

15
Display Tokens
python -m mini_js.main --tokens mini_js/examples/tc1_odd_even.js
This prints the token stream generated by the lexer.

Display AST
python -m mini_js.main --ast mini_js/examples/tc1_odd_even.js
This prints the Abstract Syntax Tree generated by the parser.

Running Unit Tests
Run all tests:

python -m pytest
Run a specific test file:

python -m pytest mini_js/tests/test_arrays.py
Run tests with verbose output:

python -m pytest -v
